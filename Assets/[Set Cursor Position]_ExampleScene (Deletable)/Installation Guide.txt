Installation Guide:

1) Api Compability Level:
Open Unity and go to Edit -> Project Settings -> Player -> Other Settings -> Api Compability Level and set it to ".NET 2.0"

2)Aplying Cursor Component
Drag the "HardwareCursor" C# script located inside of Plugins folder and drop it on any form of gameobject inside your scene.

Read the Read_Me file for more information
